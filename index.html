Введение во Frontend Разработку

Frontend разработка отвечает за создание интерфейса пользовательской части веб-сайта или веб-приложения.Она включает в себя работу с HTML, CSS и JavaScript

В начало

Языки Frontend Разработки

  • HTML (HyperText Markup Language) - используется для структуирования содержимого страницы;
  • CSS (Cascading Style Sheets) - используется для стилизации веб-страницы и создания её внешнего вида
  • JavaScript - используется для добавления интерактивности на веб-страницу, такой как анимации,обработка событий и динамическое обновление содержимого так как с помощью CSS можно создать единый формат оформления для базовых страниц, а не прописывать его многократно

В начало

Инструменты Frontend Разработки

  1. Visual Studio Code - Мощнейший редактор кода с поддержкой множества языков и плагинов
  2. React - библиотека для создания пользовательских интерфейсов
  3. Sass - репроцессор CSS с расширенными возможностями
  4. webpack - Сборщик модулей для JavaScript приложений
